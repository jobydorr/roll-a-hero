# DM Craft — Quick Card

**This is a router, not a summary.** It exists so you don't have to load `DM_CRAFT.md` (~29k words) to build something.

**How to use it:**
1. **Building something?** → find it in **§B Build Sequences**. Work the sequence.
2. **Something feels wrong?** → find the symptom in **§C Triage**. It names the section to open.
3. **Need a form to fill in?** → **§D Templates.** Most prep can be done from here alone.
4. **Want the *reasoning* or the worked examples?** → open `DM_CRAFT.md` at the section number. **Read that section only.**

> **`DM_CRAFT.md` always wins.** Nothing here is new; it's all pointers. If they ever disagree, the big doc is right and this card is stale.

## A · The Laws

The things that change decisions on their own. If you remember nothing else:

1. **Design the conditions for something memorable — don't script what happens.** *(The through-line of all 20 sources.)*
2. **Make sure every person at the table gets their thing.** A stake in the campaign, a ride in the city, a moment in the fight. *(The most-corroborated claim in the doc — 4 sources, 4 scales.)*
3. **Start from a situation, not an outcome.** "The scout has been missing three days" — not "they find the body."
4. **The DM is not telling a story. The table is.** No DMPCs. §2.1
5. **Every scene needs a value shift.** If they feel the same walking out as walking in, the scene failed. §5.0
6. **If holding back costs nothing, they'll hold back. Add a clock.** §8.5
7. **Build tension before terrain.** *What is this world arguing about?* §1.0
8. **Your world exists to challenge the characters, not to display your craft.** §1.0 cmdt. 10
9. **Prep is ammunition, not a rail.** When they go somewhere unplanned: *yes, and.* §1.9 sin 5
10. **Mention it once = atmosphere. Twice = playable. Three times = you made a promise.** §1.0 cmdt. 8
11. **Encounter ≠ combat.** Anything that taxes resources — HP, time, goodwill, trust. §5.5
12. **A "dungeon" is any bounded space + sequence of challenges + a goal.** A heist. A ball. An exam. §1.13.0
13. **Whatever they need to make an informed choice is FREE. The dice are for bonuses, not prerequisites.** §6.3
14. **Roll in the open.** If they suspect the dice don't decide, their choices stop mattering to them. §2.3

**Our table's filter (always):** would a 9-year-old get this? Family-friendly with a light edge. Spooky villains and real stakes, yes. **No perma-death** — heroes get knocked out, captured, or rescued.

## B · Build Sequences

*Work these in order. Each arrow is a decision.*

| Building… | Sequence | Full |
|---|---|---|
| **A world** | What is it *arguing about*? (one sentence) → what does it **lack**? → what did magic already **break**? → 3 specific details per faction (1 sensory, 1 behaviour, 1 object) → attach a **need** to every piece of lore → give each faction a **clock** | §1.0 |
| **A campaign** | Tone → Concept (high/low) → Setting → Conflict → **include the players** | §2.1 |
| **A campaign *idea*** | Premise (who/what/where) → **Seed Question** (philosophical, reasonable people disagree) → villain = the most persuasive *wrong* answer → every faction = a *different* answer → every PC has a personal stake → **can session one put the question in play within an hour, without naming it?** | §2.2 |
| **A town** | The **list** *is* the town: ~10 locations, each with 1 NPC + 1 line. Answer the 7 player questions → then the 8 DM questions → **give it ONE unforgettable landmark** | §1.2–1.7 |
| **A city** | Pick a **visual** theme ("the City of ___") → build **rides**, not shops → **every player gets a ride** → stop before the sewage system | §1.10 |
| **A dungeon** | Pick a **theme** (lies, death, greed) → pick a **mechanic that expresses it in the rules** → pick a **goal** (not always "kill the boss") → 4–6 encounters across combat/puzzle/roleplay, each pushing the mechanic → **end by colliding theme + mechanic** | §1.13.1 |
| **An NPC** | Want → Wound → Cover (an *active* lie) → Momentum (mid-task when they arrive) → **Ignition point** (let the players' first reaction decide what he is) | §3.2 |
| **A monster** | Role first (minion/standard/boss) → does it already exist? → compare 4–5 stat blocks at that tier → **reskin if you can** → stay inside the complexity budget → play-test | §4.5 |
| **A fight** | **What value shifts?** (opening feeling → closing feeling → the *pivot*) → why is this fight happening (a goal beyond "kill it") → frame the scene → 3 tensions in the environment → **give each PC one moment** → 3 layers of stakes → build in an escalation | §5.0, §5.2 |
| **A journey** | Is it worth running at all? → **choice layer**: points & paths, informed choice → **content layer**: close/far/very far = 1/2/3 authored events | §6.2, §6.8 |
| **A session** | Situation (not outcome) → tension pairs → soundscape → asymmetric clues → texture | §8.1 |

## C · Triage — symptom → section

**This is the main event. Find the symptom; open that section only.**

### The world / the story
| Symptom | Go to |
|---|---|
| The world feels generic | **§1.0 cmdt. 2** — define it by what it *lacks* |
| The world feels shallow / like a stage set | **§1.0 cmdt. 4** — fake the iceberg (1 sensory + 1 behaviour + 1 object per faction) |
| Magic exists but the society is bog-standard medieval | **§1.0 cmdt. 3** — let magic *break* your world |
| Nothing changes when they're away | **§8.5** — faction clocks |
| Players don't care about the world | **§1.9 sin 4** — the tourist effect. Ask each player for one contribution, then *use it* |
| Nothing they do seems to matter | **§1.9 sin 6** — consequence journal; delay payoffs 3–5 sessions |
| I'm prepping forever and never playing | **§8.6** — the pre-session-one checklist. Five items and you can start |
| I explained the mystery and they lost interest | **§1.9 sin 1** — partial answer + *new question* |
| My lore dump landed like homework | **§1.9 sin 2** — physical evidence, not exposition. History is *ammunition* |
| The campaign feels hollow / "stop the bad guy" | **§2.2** — you never found your Seed Question |
| Sessions feel disconnected | **§2.2** — ditto |
| Session one is "you meet in a tavern" | **§2.2 step 6** — the question isn't planted deep enough |

### People
| Symptom | Go to |
|---|---|
| Players forget every NPC's name | **§3.1** — you built from the outside in. Start with the **want** |
| An NPC feels like a quest dispenser | **§3.2** — give them a want + momentum |
| The villain is just an obstacle | **§4.1** — make them the most *persuasive wrong answer* |
| The villain is scary in a nightmare way, not an interesting way | **§4.1** — right about the problem, monstrous about the solution |
| A player has gone quiet / isn't invested | **§2.1 step 5** + **§2.2 step 5** — do they have a personal stake? |

### Fights
| Symptom | Go to |
|---|---|
| Combat feels like maths homework | **§5.0** — what value shifts? · then **§5.4b** — one terrain feature + one character moment |
| Every turn is the same turn | **§5.1 sin 2** — put in things to *interact* with |
| The fight is decided but still going | **§5.1 sin 3** — next hit kills; or let them flee/surrender |
| Every fight feels like every other fight | **§5.1 sin 4** — the empty room. Frame the scene; give the fight a *goal* |
| The boss died in one round | **§5.1 sin 5** — action economy. A boss is a *complete encounter* · **§4.7** minion/boss combo |
| The cleric won't heal until someone drops | **§5.1 sin 6** — make going down *cost* something |
| They nova, then rest, every single fight | **§5.1 sin 7** — no clock is running · **§8.5** |
| Tension keeps draining away after round 1 | **§5.2 P6** — escalate. Tension only falls on its own |
| Only the bard and the wizard ever matter | **§5.2 P5** — the heist-movie problem. Everyone's the getaway driver |
| Fights are exciting but nobody's scared | **§5.2 P3** — layered stakes (immediate / environmental / bigger picture) |

### Monsters
| Symptom | Go to |
|---|---|
| I forgot half its abilities | **§4.7** — the complexity budget. It only gets 3–5 turns |
| It's a bag of hit points | **§4.4** — best move ≠ usable every round; give it a *weakness to find* |
| I need a monster fast | **§4.6A** — reskin. Find its core ability, re-fiction it |
| Wrong power level for the party | **§4.6C** — power-level conversion |
| "Should it have lots of abilities or few?" | **§4.7b** — 3–5. The complexity budget *is* the variety budget |

### Places & journeys
| Symptom | Go to |
|---|---|
| The dungeon is a checklist of rooms | **§1.13.4** — most "choices" aren't choices |
| Corridors are dead space | **§1.13.7** — what *question* does this room put in their heads? |
| The trap felt unfair | **§1.13.8** — run every trap as a *puzzle*. Seed it first |
| They never retreat, it's win-or-die | **§1.13.11** — retreat must be *possible but costly* |
| The city is an endless shopping trip | **§1.10** — build rides, not shops |
| Travel is boring | **§6.1** — skip it. Or **§6.8** — TEST: close/far/very far |
| They picked a road at random | **§6.3** — informed choice. Give them the info for **free** |

### Me
| Symptom | Go to |
|---|---|
| Prep feels like homework | **§8.1 layer 4** — texture. Write something *you're* excited by |
| I'm heading for burnout | **§1.10D** — you're chasing realism. *It's a game, not a test* |
| This monster doesn't excite me | **§4.4** — then throw it out. You're a player too |

## D · Templates

*Fill these in directly. You don't need the big doc for any of them.*

### Town Sheet
```
TOWN: <name>                      Population: <n>
Reason it exists / major resource:
Culture & disposition (1 sentence):
THE LANDMARK (the one unforgettable thing):

— THE 7 PLAYER QUESTIONS —
Inn:              <name> | keeper | its one quirk
Equipment:        <shop / market / passing caravan>   cap: <pop> gp
Selling loot:     <who buys the special stuff>        cap: <pop> gp per merchant
Who's in charge:  <mayor / noble / priest / guard captain — and who hates whom>
Information:      <who knows what — and how it's telegraphed>
Healing & magic:  <who, how limited, and WHY they can't come with you>
Problems:         THE PROBLEM: ...        THE GOSSIP: ...

— LOCATIONS (aim ~10) —
1. <place> — <NPC> — <one line: hook / item / oddity>
```
*Ratios:* max item value ≈ population in gp · ~5 people per building · ~1% of the population is real defence.

### Pre-Session-One Checklist *(all you need to start)*
- [ ] **One central tension**, one sentence: *something is at stake and the people who depend on it can't agree what to do.*
- [ ] **2–4 factions with incompatible goals** — each a different answer to it
- [ ] **One limitation** that makes this world different (what's scarce / forbidden / broken / gone)
- [ ] **Three sensory details** for the starting area (one thing to smell/hear/see, one custom, one object)
- [ ] **One mystery you don't know the answer to yet**

### Campaign Launch Checklist
- [ ] Tone named in one word — **and said out loud to the table**
- [ ] Concept picked (high or low)
- [ ] Setting built outward from the concept, only as far as is live
- [ ] Conflict defined — **and the players start ON it**
- [ ] **Seed question found** (reasonable people could disagree)
- [ ] Villain = the most persuasive *wrong* answer to it
- [ ] Every faction = a *different* answer
- [ ] Every PC has a personal stake — *would they break if moved to another campaign?*
- [ ] **Session one puts the question in play within an hour, without naming it**
- [ ] Zero DMPCs

### Travel — TEST
**Close = 1 event · Far = 2 · Very far = 3.** Then they arrive. No miles. No rations.
- 🔴 **Red** = combat (authored, never rolled) · 🟡 **Yellow** = exploration/skill · 🔵 **Blue** = roleplay
- 🟣 Purple = red+blue · 🟢 Green = yellow+blue · ⚪ **White = the players choose which it becomes**

### Monster Complexity Budget
> **It lives 3–5 rounds. It gets 3–5 turns. Anything more will never happen.**

- **One really good basic attack** (make it worth using — not a fallback)
- **Plus 1–3 of:** an **opening** move · a **combo** move (conditional — vs. prone/restrained) · a **finishing** move
- **Hard caps:** ONE bonus action · ONE reaction · ONE recharge · ONE per-day *(bosses excepted)*
- **Word-count sanity check:** ~200 words = minion · ~400 = standard · 500–800 = boss
- **Best trick:** minions with ONE cool thing (*shove 5ft on a hit*) + a boss with an area attack **they're shoving people into**

### The Minimum Viable Good Fight
> **One terrain feature someone can interact with + one moment designed for a specific character.**

That's five minutes of prep and it stops a fight being maths homework.
